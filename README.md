# Ansible (ansible)
Ansible is an open-source automation platform that provides simple IT automation for cloud provisioning, configuration management, application deployment, and orchestration.

**URL:** [Visit APIs.json URL](https://www.ansible.com)

## APIs

### Ansible Automation Platform API
RESTful API for Ansible Automation Platform (formerly Ansible Tower) that enables programmatic access to job templates, inventories, credentials, and automation workflows.

**Human URL:** [https://docs.ansible.com/automation-controller/latest/html/controllerapi/](https://docs.ansible.com/automation-controller/latest/html/controllerapi/)


#### Tags:

 - Automation, Jobs, Workflows, Inventories

#### Properties

- [Documentation](https://docs.ansible.com/automation-controller/latest/html/controllerapi/)
- [OpenAPI](https://your-controller-host/api/v2/swagger/)
- [Authentication](https://docs.ansible.com/automation-controller/latest/html/controllerapi/authentication.html)

### AWX API
RESTful API for AWX (the open-source upstream project for Ansible Automation Platform) providing access to job management, inventory, and automation resources.

**Human URL:** [https://github.com/ansible/awx](https://github.com/ansible/awx)


#### Tags:

 - Open Source, Automation, Jobs, REST API

#### Properties

- [Documentation](https://docs.ansible.com/ansible-tower/latest/html/towerapi/)
- [GitHub Repository](https://github.com/ansible/awx)
- [API Explorer](https://your-awx-host/api/v2/)

### Ansible Galaxy API
API for Ansible Galaxy, a hub for finding, sharing, and downloading Ansible roles and collections.

**Human URL:** [https://galaxy.ansible.com/](https://galaxy.ansible.com/)


#### Tags:

 - Collections, Roles, Community, Content

#### Properties

- [Documentation](https://galaxy.ansible.com/docs/)
- [API Documentation](https://galaxy.ansible.com/api/v3/)
- [GitHub Repository](https://github.com/ansible/galaxy)
